Sobre

Desenvolvido em 2010 com base no trabalho de Tomi Vanek, este serviço foi minha última aplicação em Java Server Faces. Assim que sobrar tempo vou migrar para outra tecnologia.
Nessa última versão foi adicionada internacionalização ao XSLT, controle do nível de detalhe da documentação, e atualização dos estilos.
